# Manikanta Prasanth Kumar — Enhanced Portfolio v2

## 🚀 Deploy to GitHub Pages (Free, 5 minutes)

### Step 1: Create Repository
1. Go to [github.com](https://github.com) → New repository
2. Name it: `MANIKANTA131149.github.io`
3. Set to **Public** → Create repository

### Step 2: Upload Files
1. In the new repo, click **"uploading an existing file"**
2. Upload ALL files maintaining folder structure:
   ```
   index.html
   css/style.css
   js/main.js
   README.md
   ```
3. Click **"Commit changes"**

### Step 3: Enable Pages
1. Repo → **Settings** → **Pages**
2. Source: **Deploy from a branch** → **main** / **root**
3. Save → Wait 2-3 minutes

### Live at: `https://MANIKANTA131149.github.io`

## 🛠 Customization

- **Colors**: Edit `--accent`, `--gradient` in `:root` in `css/style.css`
- **Content**: All in `index.html` — find sections by `id`
- **Profile photo**: Already linked from LinkedIn. To use local photo: save as `assets/profile.jpg` and update src in index.html
